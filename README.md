# Quality Assurance Experiment Management Information System

A web application designed for gathering Software Quality Assurance Metrics in classrooms to suppport evidence based teaching. The system enables structured comparison of software development methodologies (e.g. Agile and Waterfall) and was developed as part of an MSc Software Engineering dissertation.

## Platform
- Web (Cross-platform)

## Tech Stack
- React
- JavaScript
- Node.js
- HTML / CSS

## Features
- Experiment setup and configuration
- Management of QA experiment data
- Tracking of experiment execution progress
- Results overview and basic analysis interface
- Graphs showing Quality Assurance Metrics
- Clean and user-friendly desktop UI
